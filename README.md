# ResearchSynthAI

How to use AI to run better literature searches, build systematic search strategies, synthesize evidence, and map citations around a topic.

## Topic

This talk focuses on practical AI methods for evidence discovery and synthesis.
It covers the full workflow: finding high-quality papers, designing a reproducible search strategy, synthesizing findings, and building citation/literature maps.
The goal is to help researchers move faster while improving transparency and rigor.

## Objectives

- Find a small set of high-quality, highly relevant papers quickly.
- Build a defensible, reproducible (systematic) search strategy with AI support.
- Synthesize evidence in a structured way without losing nuance.
- Understand influence and gaps using citation/literature mapping.
- Compare the best AI-enabled tools for each stage.

## Talk Questions

1. How to use AI to find a few good papers on your topic?
2. How to use AI to create a search strategy to conduct a (systematic) literature search?
3. How to synthesize evidence effectively using AI?
4. How to create a citation/literature map around a topic or paper?
5. What other AI tools can you use for literature search and synthesis?
